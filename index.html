Spin the Web (PDF).

The Spin the Web Project stewards the open-source Spin the Web framework, which offers a systematic approach to building enterprise web portals. The mission is to achieve enterprise branding (eBranding)—virtualizing an enterprise—by unifying diverse systems like ERP, CRM, MRP, BPMS, HR... into a single, consistent digital channel for all stakeholders. Rather than replacing existing systems, the framework creates a uniform user experience that seamlessly integrates with the current infrastructure, ensuring that the brand presence remains cohesive across all enterprise touchpoints.

Founder's Vision

Spin the Web grew out of frustration with the fragmentation of enterprise software. Rather than focusing on deep data-level integration, systems integration at the experience level is prioritized. The Spin the Web book describes the idea of the "virtualized enterprise": a single, coherent digital interface that adapts to each user's role—customer, employee, supplier, or partner. This approach is not about marketing claims; it is a practical way to present a consistent, role-aware digital experience across an organization's existing systems.

Spin the Web builds on fundamental web standards—HTTP, HTML, CSS, and JavaScript—leveraging them in a cohesive framework that takes the web to the next logical level. Through WBDL (Webbase Description Language), Spin the Web coordinates these standard technologies to develop complex, role-aware enterprise web portals.

In other words: Spin the Web doesn’t reinvent the web — it weaves the web’s existing building blocks into scalable, maintainable, and extensible enterprise solutions.

The Roadmap

The immediate roadmap is focused on building a solid organizational and technical project for open-source collaboration. The key objectives are:

  • Set up the Project: Establishing the organizational structure and governance model to support open-source collaboration.
  • Write the book: Creating documentation that explains the framework’s architecture and components, evolving alongside the project.
  • Release framework webspinner v1.0.0: Completing the initial setup of the core framework.
  • Provide a labs environment: Deploying a ready-to-use lab for demonstrating capabilities, experimenting, and building live portals, https://labs.spintheweb.org (Note: The portal may not be reachable).
  • Build this site: Building this site with the framework!
  • Continue discussion with Copilot Pages Spin the Web Project.
